Réalisation d'un petit outil pour transformer du texte dans une langue inventée vers des caractères spécifiques associés à la police d'écriture. 

Réalisé une première fois en C en mars 2023. Puis réécrit en Java en août 2024.

La langue inventée s'appelle l'Ancien Kamorien, c'est à destination d'histoire de fantasy.

Compiler : javac .\main.java

Lancer : java Main output
Les traductions seront directement ajoutés au fichier texte output.txt
